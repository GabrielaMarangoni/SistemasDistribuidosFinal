# Projeto Sistemas Distribuidos

## Descrição
O projeto consiste em um site que faz busca de pontos turisticos adicionados por outras pessoas. 

## Stack
- React
- NextJS
- Typescript
- Tailwind CSS

## Dependências
- Next/router
- Remix icon
- Tailwind CSS

## Executar aplicação
Comece instalando as dependências com o comando:
```
yarn 
```
Rode o frontend executando:
```
yarn dev
```
